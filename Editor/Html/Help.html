Welcome to FUDGE

This help-panel offers an introduction to the editor and how to work with it.

For an introduction to the engine visit the start page or the Wiki

  1. General interface concepts
    1. Open the various editor panels using "Edit" in the main menu or the keyboard shortcuts.
    2. A panel consists of multiple views that control specific sets of data.
    3. Hover on a view with the mousecursor to get further instructions.
    4. Control the views mostly by drag & drop and context menus (right mouseclick).
    5. Control numeric inputs by selecting digits with the mouse or the arrowkeys and use the mousewheel or arrow keys to alter. Press F2 to input a full new value or just start typing when the input field is selected.
  2. Start a project
    1. Select "New" under "Project".
    2. Choose or create an empty folder, the folder's name becomes your project's name.
    3. Review the project settings and hit "OK".
    4. The project panel and the graph panel open and display the current state of your new project (which is empty).
  3. Project Panel
    1. External - View the files in the project. Serves as a starting point to use external files like images or sounds via drag&drop to the view "Internal".
    2. Internal - View the internal resources of the project that will be serialized to the file "Internal.json". Drag & drop internal resources to the appropriate locations to use them.
    3. Preview - Renders a preview of the resource selected in the views "Interal" or "External".
    4. Properties - Displays the properties of the selected resource. Offers drop-locations for internal resources.
    5. Script - Shows a list of custom scripts available in the project. Drop on "Components" to attach to a node.
  4. Graph Panel
    1. Render - Drop a graph from "Internal" here to set the graph to edit with this panel. Navigate the editor camera to view the rendering of the graph. Select nodes by clicking and use the keyboard together with the mouse to manipulate transformations.
    2. Hierarchy - View the graph as a hierarchical tree. Create new nodes and manipulate the parent child relationsships
    3. Components - View the components attached to the selected node. Manipulate by adjusting values, dropping internal resources or using mouse-movement in the Render-View.
